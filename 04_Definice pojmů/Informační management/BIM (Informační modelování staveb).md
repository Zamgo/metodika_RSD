# Definice

Užívání sdílené digitální reprezentace vystavěného aktiva k usnadnění procesů navrhování, výstavby a provozu pro vytváření spolehlivého základu pro rozhodování.


## Praktický význam pro správu informací o stavbě / ŘSD

- BIM je metoda práce s informacemi o stavbě; [[Informační model stavby]] a [[Digitální model stavby (DiMS)|DiMS]] jsou jejími klíčovými výstupy.
- Realizace BIM vyžaduje [[Společné datové prostředí (CDE)]], [[Požadavky objednatele na výměnu informací (EIR)|EIR]] a [[BIM Execution Plan (BEP)|BEP]].
