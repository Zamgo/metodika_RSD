# Definice

Informační model vztahující se k dodací fázi (navrhování a provádění stavby).

*zdroj: ČSN EN ISO 19650*

## Praktický význam pro správu informací o stavbě / ŘSD

- PIM odpovídá fázi přípravy a realizace; po předání stavby může být obsah předán do [[Informační model aktiva (AIM)|AIM]] pro provoz a údržbu.
- V metodice ŘSD se zpravidla pracuje s jednotným pojmem [[Informační model stavby]]; PIM/AIM slouží pro pochopení dělení dle norem.
