# Popis

Mobilizace IT je souhrnná činnost, která zastřešuje veškeré technologické přípravy [[Realizační tým|realizačního týmu]]. Zahrnuje pořízení a instalaci software a hardware, konfiguraci [[Společné datové prostředí (CDE)|CDE]], testování konektivity a ověření funkčnosti výměn informací.

Cílem je, aby na konci mobilizace bylo IT prostředí plně funkční a otestované -- od autorských nástrojů (CAD, BIM software) přes CDE nástroj až po síťovou infrastrukturu a přístupy.

# Vstupy
- [[BIM Execution Plan (BEP)|BEP]] -- rozpis IT nasazení
- Požadavky na software a hardware z nabídkové fáze

# Výstupy
- Funkční IT prostředí realizačního týmu
- Protokoly o testování

# Kontrolní body pro správce stavby
- Ověřit, že IT prostředí odpovídá požadavkům BEP
- Zkontrolovat, že všechny [[Úkolový tým|úkolové týmy]] mají přístup k CDE
