# Popis

Po implementaci se [[Společné datové prostředí (CDE)|CDE]] konfiguruje -- struktura složek, přístupová oprávnění, stavy [[Informační kontejner|informačních kontejnerů]], schvalovací workflow a metadata.

Konfigurace musí odpovídat informačnímu standardu a metodám projektu.

# Vstupy
- Implementované CDE
- Informační standard

# Výstupy
- Nakonfigurované CDE

# Kontrolní body pro správce stavby
- Ověřit soulad s standardem
- Otestovat workflow
