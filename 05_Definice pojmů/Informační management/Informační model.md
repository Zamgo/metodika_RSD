# Definice

Množina strukturovaných a nestrukturovaných informačních kontejnerů.


## Praktický význam pro správu informací o stavbě / ŘSD

- [[Informační model stavby]] je konkrétní případ informačního modelu v kontextu stavby; v normě ČSN EN ISO 19650 se dále rozlišuje [[Projektový informační model (PIM)|PIM]] a [[Informační model aktiva (AIM)|AIM]].
